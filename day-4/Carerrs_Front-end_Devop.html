Careers @ Cricbuzz

We're building a cricket product for the generation of always connected internet users and we need your talent and passion.

Front-end developer

Cricbuzz, the most popular destination for Cricket in India, provides all round coverage of international, domestic and T20 cricket tournaments from around the world. Alongside its extremely popular website and mobile site, Cricbuzz's mobile applications for Android, iOS, Windows Phone and Blackberry are the no. 1 cricket apps on the respective app stores.


We are looking for experienced front-end developers who can help us create the best cricket experience for desktop and mobile web users.

    Responsibilities
  • Build scaleable, responsive, optimized and elegant web user experiences using the latest standards and best practices in HTML5, CSS, and JS.
  • Ownership of all front-end development, including code structure, style guides, testing, etc.
  • Improve the quality, performance, and stability of front-end codebase.
  • Work closely with product management, design, development and sales teams to create new features and refine existing ones.
  • Build reusable code for future use.
  • Make sure web pages are optimized for best speed and scalability.
    Requirements
  • 1-3 years front end development experience.
  • Be proficient in coding languages such as HTML, CSS, JavaScript, JQuery and AngularJS experience.
  • Experience with CSS preprocessors such as SASS, LESS, etc.
  • Understanding of mobile development best practices (responsive design, progressive enhancement, graceful degradation etc).
  • Understanding the principles of SEO.
  • Good problem solving skills.
